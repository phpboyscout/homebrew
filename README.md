# Phpboyscout Homebrew

## How do I install these formulae?

`brew install phpboyscout/homebrew/<formula>`

Or `brew tap phpboyscout/homebrew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "phpboyscout/homebrew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
